# ARTI308 - Lab 2

## Dataset Summary

**Dataset:** Heart Disease Dataset (from Kaggle - David Lapp)

**Shape:** 1025 rows, 14 columns

**Problem Type:** Classification (Binary)

**Target Variable:** target (1 = has heart disease, 0 = no heart disease)

**Goal:** Predict whether a patient has heart disease based on medical features such as age, sex, cholesterol, blood pressure, and other clinical measurements.
